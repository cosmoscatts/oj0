# v0.1.4

构建于 2023/12/25 晚上

## 新特性

- 【历史提交】页面新增跳转功能，可跳转至做题页面
  - 点击的是自己的提交记录，可查看提交的代码和判题信息

  - 点击的是其他人的提交记录，则等同于做题功能

- 【做题】页面子菜单【提交记录】新增点击可查看代码、判题信息功能   

## BUG 修复

- 修复上传头像报错的问题
